# Flask API Development
## Create Python Virtual Enviournment
```
virtualenv venv <--python=python3.5>
```

## Activate in windows
```
cd ./venv/Scripts/activate.bat
```


## All enviourment list
```
conda info --envs
```

## all librabry installed in Enviournment
```
pip freeze
```

## Activate Env
```
activate <Env_name>
```

## Deactivate Env
```
deactivate <Env_name>
```
